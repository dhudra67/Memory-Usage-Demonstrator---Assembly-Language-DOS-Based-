A simple assembly language project that shows how memory allocation works visually.
Author: M Faizan Ali 
What it does
Shows 10 memory blocks as [ ] (free) or [X] (used). You can allocate and free memory blocks interactively.
How to use
Compile:
bashMASM memory.asm;
LINK memory.obj;
Run:
bashmemory.exe
Commands:

a - allocate memory
f - free memory
q - quit

Requirements

MASM assembler
DOS or DOSBox
